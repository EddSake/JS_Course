JavaScript Basics - Exercises

Exercise 1:
- Create a simple "Hello, World!" program
- Add two numbers and display the result
- Use basic console.log() to output the results

Exercise 2:
- Create variables with different data types
- Perform basic arithmetic operations
- Display the results and the type of each variable

Exercise 3:
- Create a basic calculator that adds, subtracts, multiplies, and divides two numbers
- Use template literals to display the results
- Add comments explaining your code
